1- Download the required packages with python -m pip install -r requirements.txt
2- Create .env file and insert the following:
TOKEN=
SCHEDULE_URL=
RATTRAPAGE_URL=
3- Run python manage.py migrate
4- Run python manage.py runserver
5- go to /update_schedules route everytime you open the website to get the latest rattrapage schedules.
6- Done! You can view the schedules.
