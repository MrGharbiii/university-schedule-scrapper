1- Create .env file and insert the following:
TOKEN=
SCHEDULE_URL=
RATTRAPAGE_URL=
2- Run python manage.py migrate
3- Run python manage.py runserver
4- go to /update_schedules route
5- Done! You can view the schedules through SQLite Viewer extension in VS Code or go /admin route and enter
the super user credentials but first you have to create the super user with "python manage.py createsuperuser"
